Welcome to **Receipt Tracker**, a comprehensive, multi-pane application that automates the entire workflow of processing receipt images. It reads receipt images, performs OCR, queries a powerful AI model for structured JSON data, and provides an immersive D3.js dashboard for financial tracking and analytics.

---

## Quick Start

To start trying Receipt Tracker today:
1. **Download the Repository**: Clone or download this repository directly into any folder inside your Obsidian vault.
2. **Install Datacore**: Ensure you have the **Datacore** plugin installed and enabled in Obsidian.
3. **Open the Entry Note**: Open the **`RECEIPT TRACKER.md`** note inside Obsidian to launch the component!

---

## Features

### End-to-End Automated Workflow
*   **Image Processing**: Scans a user-configurable folder in the vault for receipt images (.png, .jpg, etc.).
*   **OCR Text Extraction**: Uses the **Tesseract.js** library to perform local OCR on a selected image, including a language selector and auto-detect.
*   **AI-Powered Data Parsing**: Sends the raw text to the **Groq API** (Llama 3) to parse and return a structured JSON object.
*   **Data Persistence**: Creates a new, organized markdown file for each processed receipt containing the extracted data in frontmatter and a JSON code block.

### Interactive Processor UI
*   **Multi-Pane Layout**: Features a professional, resizable interface including a file list, a processing panel, and a summary table.
*   **Focus Mode**: Each panel can be individually expanded to fill the screen.
*   **Manual Editing**: If the AI makes a mistake, users can easily manually correct the extracted JSON data.

### Full Financial Dashboard
*   **Data Aggregation**: Switches to a "Dashboard" view that aggregates data from all processed receipt files.
*   **Live Currency Conversion**: Fetches real-time exchange rates to convert all amounts to a single base currency for analysis.
*   **Rich Visualizations**: Uses **D3.js** to render interactive charts, including a "Monthly Spending" bar chart and a "Top Spending" pie chart.
